# ESP32 S3 WROOM1 tester

Small test to make sure you did not receive a dead board.

You will need esp32 and adafruit neopixel libraries.
Replace with your Wi-Fi credentials, upload sketch to your board, open serial, copy IP and go to this web page:

http://your_ip/

You should be able to turn the internal LED on and off, it'll glow green.
